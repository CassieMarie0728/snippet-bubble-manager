# Add Smart Folders, Presets, and Metadata Features with Bubble Minimize

This PR introduces a significant feature expansion:

## New Features
- **Minimize Bubble**: Collapse the main app to a floating bubble widget for quick access
- **Smart Folders**: Rule-based dynamic folders that filter snippets by tags, labels, languages, frameworks, platforms, and complexity
- **Presets**: Save and reuse snippet creation templates with pre-configured metadata
- **Enhanced Metadata**: Add labels, platform, framework, and complexity flags to snippets
- **Folder Customization**: Choose colors and icons for regular folders
- **Sorting Options**: Sort snippets by added date, updated date, name, favorites, or custom order

## Improvements
- Refactored modal components into separate, reusable files
- Updated Firestore security rules for new collections
- Enhanced export/import to preserve smart folders and presets
- Added recent tags/labels suggestions in the snippet editor
- Improved drag-snap logic for bubble positioning
- Better visual organization with metadata badges

## Breaking Changes
- Export format now includes version field and new collections (smartFolders, presets)
- Legacy imports (array format) are still supported for backward compatibility

## Files Changed
- Core component refactoring: SnippetCard, SnippetModal, SmartFolderModal, PresetModal
- Type definitions updated for new data structures
- Utility functions extracted (cn helper)
